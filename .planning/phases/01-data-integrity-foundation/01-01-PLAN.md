<threat_model>
ASVS level: 1
Block on: high severity

- **TM-01: App Group state drift.** `TimerState` is shared with the widget. Mitigation: keep persisted `goal` and `danger` in the existing `TimerState` payload and avoid adding sensitive fields.
- **TM-02: Notification surprise.** Scheduling reminders by default could surprise users. Mitigation: default reminder remains `0 minutes`, and nonzero reminders are opt-in.
- **TM-03: Preference inconsistency.** Different controls writing different reminder values can create unexpected notifications. Mitigation: both controls read from `AppConfig.Timer.reminderOptions`.
</threat_model>
